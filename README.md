# 🎬 CineMind AI

AI-powered movie recommendation dashboard built using Spring Boot, JavaScript, HTML, CSS, and TMDB API.

## 🚀 Live Demo

**Frontend:** https://cinemind-ai-la2w.onrender.com

**Backend API:** https://cinemind-qzpm.onrender.com

---

## 📌 Features

* Search movies by name
* View movie details
* Watch movie trailers
* AI-powered movie recommendations
* Trending movies section
* Add and remove favorite movies
* Dark Mode / Light Mode toggle
* Responsive user interface
* Real-time TMDB API integration

---

## 🛠️ Tech Stack

### Frontend

* HTML5

* JavaScript

### Backend

* Spring Boot
* Java
* Maven

### API

* TMDB (The Movie Database) API

### Deployment

* Render

---

## ⚙️ How It Works

1. User searches for a movie.
2. Frontend sends a request to the Spring Boot backend.
3. Backend fetches movie data from TMDB API.
4. Movie details are displayed on the dashboard.
5. Similar movies are recommended based on genre and language.
6. Users can save movies to Favorites.
